# 🧸 更多物品&配方

### 系统介绍

该系统扩充了原版MC的方块、矿物、装饰品、装备...等

快捷指令：

```
/ia - 打开更多合成物品列表
```

点击对应选项菜单可查看物品配方、使用方式、功能
